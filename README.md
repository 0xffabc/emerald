## Features

- Give weapons
- Several weapon exploits (rapid fire, infinity, invisible fire)
- Physical immortality
- Set random HP to every player (for educational purposes)
- Communism: You can affect other players with your hacks, from giving away impulse guns to making others immortal
- Backtrack: Become the most annoying sniper in war4, that manages to kill everybody even before they see you
- Weapon merger: Test a symbiosis of impulse gun and bazooka against unsuspecting players
- Custom material ID for cube guns: Trap players in a box of kill block (because it's fun)
- Keybinds: Become a proficient hacker
- Theme customization

## Install

Go to [releases](/releases) and install `Ethyr-final.user.js` as a userscript, using any of these managers:

- [Violentmonkey](https://violentmonkey.github.io/) (preferred, open-source)
- Tampermonkey (closed-source)

## Setup

Install the dependencies:

```bash
npm install
```

## Development

Start the dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
